# Open OSRS External Plugins

This repository contains external plugins for the Open OSRS RuneScape client. You can install these plugins by adding this repository (toasty-toast/open-osrs-external-plugins) in your Open OSRS client.

## Plugins

|Plugin|Description|
|------|-----------|
|Bank PIN|Highlights the next number you need to click when entering your bank PIN.|
|Group Iron Panel|Adds a new side panel that shows your group iron group members' stats. After installing the plugin, open the Group Iron tab in game to allow the plugin to detect your group name and members.|
